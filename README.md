# IMDB-Excel-Data-Analysis
IMDB movie dataset analysis using Excel including data cleaning, formulas, pivot tables, and dashboard creation.

## Project Overview
This project analyzes the IMDB movie dataset using Microsoft Excel.

The analysis includes:
- Data Cleaning
- Helper Columns
- Formula-based Analysis
- Pivot Table Analysis
- Interactive Dashboard
- Business Insights

## Tools Used
- Microsoft Excel
- Pivot Tables
- Excel Formulas
- Data Visualization

## Key Analysis
- Top 10 movies by IMDb score
- Top 10 movies by gross revenue
- Genre performance analysis
- Profit analysis
- Correlation between budget and revenue

## Dashboard
The dashboard includes:
- KPI cards
- Revenue trend by year
- Genre distribution
- Top movies by revenue

## Business Insights
Some insights discovered:
- Certain genres generate higher revenue.
- High budget movies tend to produce higher gross revenue.
- Specific directors consistently produce high-rated films.
